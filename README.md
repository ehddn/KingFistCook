+ ## **게임 상세**
  + ### 개요
    + #### 게임잼 키워드 '주먹'을 주제로 주먹으로 음식을 요리하는 타이쿤 게임을 생각하여 제작했습니다.
    + 주먹을 내리쳐 재료를 손질, 밥을 짓는 등 순서대로 음식을 요리하며 타이밍에 맞추어 버튼을 누르는 간단한 조작으로 이루어져 있습니다.
  + ### 핵심 플레이
    + #### 주먹밥, 연어초밥, 돈까스 차례차례 메뉴판에 적힌 음식들을 요리해 나가자
